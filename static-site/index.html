KEYSPY Documentation

Introduction

KEYSPY is an AI-powered SEO and SEM platform built with Next.js and Tailwind CSS. It helps users analyze their website for SEO gaps, discover missing keywords, generate SEO-optimized content, and get ad copy suggestions based on competitor analysis.

Core Features

  • SEO Audit Module: Analyze websites for SEO opportunities
  • AI Content Generator: Create SEO-optimized articles
  • Ad Intelligence: Get ad copy suggestions based on competitor analysis
  • Dashboard View: Track all actions in a timeline
  • User Authentication: Email and Google login
  • Stripe Billing: Free/Pro/Enterprise plans

Customization

KEYSPY includes a built-in theme customizer that allows you to change the visual appearance of your application in real-time:

  • Brand Name: Change the displayed name throughout the platform
  • Primary Color: Choose from several color themes
  • Light/Dark Mode: Switch between color schemes
  • Font Family: Change typography throughout the application
  • Border Radius: Adjust how rounded UI elements appear

Deployment Guide

Deploying KEYSPY

Prerequisites

  • A Vercel account (free tier is available)
  • Your KEYSPY codebase in a Git repository
  • Completed any customizations you want for your initial deployment

Deployment Steps

  1. Log in to Vercel
  2. Import your project from Git repository
  3. Configure project settings
  4. Set up environment variables (if needed)
  5. Click "Deploy"

Setting Up a Custom Domain

To use your own domain name instead of the Vercel-provided URL:

  1. Go to your project in the Vercel dashboard
  2. Click on "Settings" > "Domains"
  3. Enter your domain name and click "Add"
  4. Follow Vercel's instructions to update your DNS settings

Customization Guide

Customizing KEYSPY

KEYSPY is designed to be easily customizable without coding knowledge. Here's how to customize various aspects of the platform:

Theme Customization

KEYSPY includes a built-in theme customizer that allows you to change the visual appearance without editing code:

  1. Access Theme Customizer by clicking the settings icon in the bottom-right corner
  2. Customize Brand Name, Primary Color, Mode, Font Family, and Border Radius
  3. Click "Apply Changes" to save your customizations

Text Content Customization

All text content in KEYSPY can be easily modified through a central configuration file:

  1. Navigate to the src/config.ts file in your project
  2. Edit the text values in this file to update content throughout the application
  3. Save the file and rebuild/redeploy the application
